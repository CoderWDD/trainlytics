## 一、今日记录模块（Today）

### 1.1 今日总览仪表盘
- [x] 展示当前阶段标签（减脂期/增肌期等）
- [x] 展示今日完成进度（已记录项/总项数）
- [x] 展示今日身体数据摘要（体重、体脂、BMI）
- [x] 展示今日营养摄入摘要（早/午/晚/加餐热量）
- [x] 展示今日训练状态（训练名称、动作数、已完成组数）
- [x] 展示每日笔记区域
- [x] FAB 快速添加按钮（记录体重 / 添加餐食 / 开始训练）
- [x] 毛玻璃效果底部导航栏

### 1.2 身体数据记录
- [x] 横向刻度尺选择器记录体重（kg）
- [x] 横向刻度尺选择器记录体脂率（%）
- [x] 横向刻度尺选择器记录腰围（cm）
- [x] 带时间戳保存记录

- [x] 底部弹窗：应用餐食模板

### 2.2 AI食物识别
- [x] 拍照/文字描述识别食物种类
- [x] 展示识别结果（食物名称 + 宏量信息）
- [x] 支持微调识别结果中的营养数值
- [x] 一键确认并添加到餐食记录
- [ ] 支持扫描食品条形码识别（待补充）

### 2.3 手动营养录入
- [x] 输入食物名称

### 2.4 食物数据库
- [x] 食物搜索（关键词搜索）

- [ ] 自定义食物管理（创建/编辑/删除）（待补充）

## 三、训练追踪模块（Workout）

### 3.1 开始训练
- [x] 从已保存训练模板中选择并开始
- [x] 开始空白训练会话
- [x] 入口跳转创建新训练模板

### 3.2 训练进行中
- [x] 训练计时器（实时计时）
- [x] 动作卡片列表，支持记录每组的重量/次数/RPE
- [x] 添加新组数功能
- [x] 动态添加训练动作（通过动作选择器）
- [x] 实时计算估算训练量（lbs/kg）

### 3.3 训练完成总结
- [x] 训练完成总结页（本次时长/训练量/PR提示/疲劳度评估）

### 3.4 动作选择器
- [x] 动作关键词搜索
- [x] 按肌群分类筛选（全部/胸部/背部/腿部/肩部/手臂/核心）
- [x] 多选动作批量添加
- [x] 动作卡片展示（名称 + 图片）

### 3.5 力量进展追踪
- [x] 记录每次训练的重量×次数数据
