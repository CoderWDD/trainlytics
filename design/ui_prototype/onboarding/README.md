# 引导/欢迎页

所属模块: 系统 (System)
页面类型: 独立页面（多步骤流程）

## 页面说明

首次使用引导流程，共4步：品牌欢迎 → 身体基础数据采集 → 目标选择 → 训练经验水平。仅在首次启动时显示，完成后跳转今日总览。

## 关键组件

- 步骤指示器（4步进度点）
- 步骤1: 品牌欢迎（Logo + 标语 + 特性亮点列表）
- 步骤2: 身体数据采集（身高/体重/性别/年龄输入，RulerSlider）
- 步骤3: 目标选择（减脂期/增肌期/维持期，单选卡片）
- 步骤4: 训练经验水平（初学者/中级/进阶，配训练频率说明）
- 主色渐变 CTA 按钮（继续/完成）
- 跳过入口（右上角文字链接）

## 入口

- 首次启动应用

## 出口

- 完成引导 → today_dashboard
- 跳过 → today_dashboard（使用默认值）
