## 1. 应用整体架构

应用采用 **底部 Tab 导航** 模式，共 4 个主 Tab 页。设置页面不属于 Tab 导航，通过个人头像/更多入口进入。

### 底部导航栏（4 个 Tab）

| Tab | 中文名 | 英文标识 | 说明 |
|-----|--------|---------|------|
| 1 | 今日 | Today | 当日数据总览与快速操作 |
| 2 | 历史 | History | 历史记录浏览与补记 |
| 3 | 洞察 | Insights | 数据分析与趋势可视化 |
| 4 | 模板 | Templates | 训练/餐食模板管理 |

### 非导航页

- **设置（Settings）**：通过个人头像或"更多"入口进入，非底部 Tab

## 5. 各 Tab 详细流程

### Tab 1：今日（Today）

入口：底部导航 "今日" Tab

```
today_dashboard（今日主页面）
│
├── [FAB "+"] 按钮
│   └── quick_add_sheet（快速添加底部弹窗）
│       ├── 记录体重 → record_body_stats（身体数据记录）
│       ├── 添加餐食 → record_meal_sheet（餐食记录页）
│       └── 开始训练 → start_workout_sheet（选择训练模板弹窗）
│
├── "继续训练" 按钮（有进行中训练时显示）
│   └── active_workout（正在训练）
│
├── "[记录]" 按钮
│   └── record_body_stats（身体数据记录）
│
└── 营养卡片（点击可展开/编辑）
    └── record_meal_sheet（餐食记录页）
```

**核心用户流程**：打开 App → 一目了然今日概况 → 通过 FAB 快速添加各项记录

---

### Tab 2：历史（History）

入口：底部导航 "历史" Tab

```
history_list（历史记录主页面）
│
├── 点击某天卡片
│   └── history_day_detail（单日详情）
│       └── 查看该日训练、饮食、身体数据详情
│
└── "补记日志" 按钮
    └── backfill_journal（补记日志）
        ├── 补记体重 → record_body_stats
        ├── 补记餐食 → record_meal_sheet
        └── 补记训练 → start_workout_sheet
```

**核心用户流程**：浏览历史记录 → 查看某日详情 → 或对遗漏数据进行补记

---

### Tab 3：洞察（Insights）

入口：底部导航 "洞察" Tab

```
insights_dashboard（数据分析主页）
│
├── 体重趋势图表
├── 训练频率统计
├── 营养摄入分析
│
└── "每周回顾" 按钮
    └── weekly_review（每周回顾）
        └── 查看本周训练总结、体重变化、营养概况
```

**核心用户流程**：查看各项数据趋势 → 点击每周回顾获取周度总结

---

### Tab 4：模板（Templates）

入口：底部导航 "模板" Tab

```
template_gallery（模板管理主页）
│
├── 训练模板卡片（点击）
│   └── workout_template_editor（训练模板编辑）
│       ├── "添加动作" → exercise_picker（选择动作弹窗）
│       └── "保存" → template_gallery
│
├── 餐食模板卡片（点击）
│   └── meal_template_editor（餐食模板编辑）
│       ├── "添加食物" → food_search_sheet（食物搜索弹窗）
│       │     ├── "+"添加 → 返回 meal_template_editor
│       │     ├── "手动输入" → manual_food_entry
│       │     └── "AI识别" → ai_food_recognition
│       └── "保存" → template_gallery
│
└── "新建模板" 按钮
    └── workout_template_editor（训练模板编辑）
```

**核心用户流程**：浏览已有模板 → 编辑/管理模板 → 创建新模板

---


## 6. 子页面流程

### 6.1 正在训练（active_workout）

```
active_workout（正在训练）
│
├── 当前动作组列表（可编辑组数、重量、次数）
├── 动作间切换
│
├── "添加动作" 按钮
│   └── exercise_picker（选择动作弹窗）
│       └── 选择动作 → 返回 active_workout
│
└── "完成训练" 按钮
    └── today_dashboard（返回今日主页）
```

---

### 6.2 餐食记录（record_meal_sheet）

```
record_meal_sheet（餐食记录页）
│
├── 已记录餐食列表
│
├── AI拍摄入口
│   └── ai_food_recognition（AI识别结果）
│       ├── 显示识别出的食物及营养数据
│       ├── 确认 → 返回 record_meal_sheet
│       └── 手动调整 → 修改后返回
│
└── 手动输入入口
    └── manual_food_entry（手动录入营养）
        ├── 搜索/选择食物
        ├── 输入份量
        └── 完成 → 返回 record_meal_sheet
```

---

### 6.3 开始训练（start_workout_sheet）

```
start_workout_sheet（选择训练模板弹窗）
│
├── 选择已有模板
│   └── active_workout（开始训练）
│
├── "开始空白训练"
│   └── active_workout（开始空白训练）
│
└── "创建新模板"
    └── workout_template_editor（训练模板编辑）
        └── 保存后可开始训练
```

---

### 6.4 补记日志（backfill_journal）

```
backfill_journal（补记日志）
│
├── 选择日期
│
├── "补记体重"
│   └── record_body_stats（身体数据记录）
│
├── "补记餐食"
│   └── record_meal_sheet（餐食记录页）
│
└── "补记训练"
    └── start_workout_sheet（选择训练模板弹窗）
```
