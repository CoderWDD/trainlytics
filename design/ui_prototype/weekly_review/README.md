# 每周回顾

所属模块: 洞察 (Insights)
页面类型: 子页面

## 页面说明

每周表现综合回顾页，展示整体评分、身体变化、营养平均值、训练1RM估算和下周目标，帮助用户全面评估本周表现并规划下周重点。

## 关键组件

- 头部"每周回顾"+日期范围+"进展良好"状态
- Bento网格: 表现评分92/100, 身体变化(平均体重-0.8 lbs, 柱状图), 营养(平均2,450 kcal, 平均192g蛋白质, 6/7目标达成)
- 训练表现: 5/7频率, 42.8k kg总训练量, 1RM估算卡(深蹲315 lbs +12, 卧推225 lbs, 硬拉405 lbs +5)
- 周笔记区+"下周重点"清单

## 入口

- insights_dashboard"每周回顾"按钮

## 出口

- 返回 → insights_dashboard
- 底部导航切换
