# 训练完成总结页

所属模块: 今日 (Today)
页面类型: 子页面（全屏）

## 页面说明

训练完成后自动跳转，展示本次训练的完整总结：时长、总训练量、PR提示、疲劳度评估和与历史的对比。是训练流程的闭环节点。

## 关键组件

- 头部庆祝动效区（"训练完成！" + 本次训练名称）
- Bento网格统计卡片组：
  - 训练时长卡（大字体，如 "47:32"）
  - 总训练量卡（如 "12.4k kg"）
  - 完成组数卡（如 "18 组"）
  - 最高强度卡（估算1RM）
- PR 提示横幅（金色高亮，如 "🏆 深蹲新PR：315 lbs"）
- 疲劳度评估（1-5星评分组件）
- 本次 vs 上次对比区（训练量/时长增减百分比）
- 训练笔记输入框
- 底部双按钮：继续→今日主页 / 查看详情→history_day_detail

## 入口

- active_workout "完成训练" 操作

## 出口

- "返回今日" → today_dashboard
- "查看详情" → history_day_detail
