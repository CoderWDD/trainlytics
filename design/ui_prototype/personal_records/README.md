# 个人记录（PR）专页

所属模块: 训练 / 洞察
页面类型: 子页面

## 页面说明

集中展示所有训练动作的个人最佳成绩（PR），含1RM估算、实际最佳重量×次数记录和PR创建时间线。帮助用户追踪力量进步。

## 关键组件

- TopAppBar（返回 + "个人记录"标题 + 搜索按钮）
- 精英徽章卡（总PR数 + 本月新PR数）
- 动作搜索栏 + 肌群分类 Tab（全部/胸/背/腿/肩/臂/核心）
- 动作PR卡片列表（每卡含）：
  - 动作名称 + 肌群标签
  - 最佳1RM估算（大字体）
  - 最佳单组：重量 × 次数 + 创建日期
  - 小折线图（近6次该动作的1RM趋势）
  - PR标记徽章（新PR高亮）
- 空状态：引导用户开始第一次训练

## 入口

- insights_dashboard 侧边入口
- settings 菜单项
- history_day_detail PR徽章点击

## 出口

- 返回 → 上一页
- 点击动作卡 → 展开历史详情（内联展开）
