# 开始训练

所属模块: 今日 (Today)
页面类型: 底部弹窗 (Bottom Sheet)

## 页面说明

开始训练选择弹窗，用户可从已保存的训练模板中选择，或开始空白训练会话。

## 关键组件

- 模糊仪表盘背景
- 底部弹窗: "开始训练"标题，关闭按钮
- "已保存模板"区: 推日（胸/肩/三头，6动作），拉日（背/二头/后束，5动作），腿日（股四/腘绳/小腿，7动作）
- "开始空白训练"渐变按钮
- "创建新模板"次要按钮

## 入口

- `quick_add_sheet` "开始训练"
- `today_dashboard` 直接入口

## 出口

- 选择模板 → `active_workout`
- "开始空白训练" → `active_workout`
- "创建新模板" → `workout_template_editor`
