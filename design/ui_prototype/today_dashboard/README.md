# 今日总览

所属模块: 今日 (Today)
页面类型: 主页面 (Tab页)

## 页面说明

今日总览仪表盘，以Bento网格布局集中展示当日训练阶段、身体数据、餐食摄入、训练进度和每日笔记。是用户打开应用后看到的第一个页面。

## 关键组件

- 品牌头部 + 头像
- Bento网格:
  - 每日摘要卡（当前阶段"减脂期"，2/4已记录）
  - 身体数据卡（体重75.2kg，体脂18.5%，BMI 23.1）
  - 营养区（早餐520kcal，午餐/晚餐/加餐位）
  - 训练卡（"推力日训练"，5个动作，15组已完成）
  - 每日笔记
- FAB "+"按钮
- 毛玻璃效果底部导航栏

## 入口

- 底部导航"今日"Tab
- 从"完成训练"返回

## 出口

- FAB "+" → `quick_add_sheet`
- "继续训练" → `active_workout`
- "[记录]" → `record_body_stats`
- 营养卡 → `record_meal_sheet`
- 底部导航切换
