# 数据分析

所属模块: 洞察 (Insights)
页面类型: 主页面 (Tab页)

## 页面说明

数据分析主仪表盘，展示体重趋势、营养趋势、训练进展的图表分析，以及每周回顾入口，是用户查看长期数据和趋势的核心页面。

## 关键组件

- 头部"数据分析"
- 概览条: 7日体重变化(-0.6 kg), "进展良好"
- 体重趋势卡: SVG折线图(30天数据, 7日移动平均线, 当前75.2 kg)
- 营养趋势: 卡路里摄入柱状图(平均2,150 kcal), 蛋白质摄入环形图(85%达成, 152g)
- 训练进展: 深蹲(120kg x5, 估算1RM 135kg), 硬拉(160kg x3, 估算1RM 178kg)
- "每周回顾"CTA

- 底部导航"洞察"Tab

## 出口
