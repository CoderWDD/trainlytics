# 通知设置页

所属模块: 设置 (Settings)
页面类型: 子页面

## 页面说明

统一管理所有推送提醒配置，包括餐食记录提醒、训练提醒、水分摄入提醒和每周回顾通知。每类通知可独立开关并设置时间。

## 关键组件

- TopAppBar（返回 + "通知设置"标题）
- 通知总开关（系统级，置顶卡片）
- 餐食提醒区块（早/中/晚餐时间设置，各有开关+时间选择器）
- 训练提醒区块（开关 + 时间选择器 + 训练日自动检测说明）
- 水分提醒区块（开关 + 每日开始/结束时间 + 间隔频率选择）
- 每周回顾提醒（开关 + 星期几选择 + 时间选择器）
- 权限状态提示卡（系统通知权限未开启时的引导）

## 入口

- settings "通知与提醒" 菜单项

## 出口

- 返回 → settings
- 系统权限引导 → 跳转系统设置（Android 系统级跳转）
