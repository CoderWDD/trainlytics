# 目标设置页

所属模块: 设置 (Settings)
页面类型: 子页面

## 页面说明

配置用户的健康目标，包括体重目标、每日营养目标（热量/蛋白质/碳水/脂肪）、训练目标（频率/每周训练量）和目标周期。作为洞察模块数据对比的基准线。

## 关键组件

- TopAppBar（返回 + "目标设置"标题）
- 体重目标卡（当前体重 → 目标体重，时间线滑块）
- 营养目标区块（热量/蛋白质/碳水/脂肪 数字输入，含推荐值提示）
- 训练目标区块（每周训练天数选择器 1-7，单次训练时长）
- 目标周期设置（4周/8周/12周/自定义）
- 阶段标签选择（减脂期/维持期/增肌期）
- 保存按钮（主色渐变）

## 入口

- settings "目标设置" 菜单项
- onboarding 步骤3完成后

## 出口

- 保存 → settings（或 today_dashboard，来自 onboarding）
- 返回 → settings
