# 设置

所属模块: 设置 (Settings)
页面类型: 主页面 (非Tab页)

## 页面说明
应用设置页，配置度量单位、默认值、日期时间偏好和数据管理选项。非底部导航Tab，从个人资料或更多菜单进入。

## 关键组件
- 头部"设置"+"精细调整您的性能实验室环境"副标题
- 单位设置: 体重单位(kg), 距离(cm/km)
- 默认值: 当前目标(减脂期), 训练模板(训练日)
- 日期时间: 周起始日(周一), 24小时制开关
- 数据管理: "导出数据"按钮
- 底部: Trainlytics v1.0.0, 隐私政策, 服务条款

## 入口
- 个人资料/更多菜单 (非Tab导航)

## 出口
- "导出数据" → export_data
- 返回
