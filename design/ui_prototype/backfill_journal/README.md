# 补记日志

所属模块: 历史 (History)
页面类型: 子页面

## 页面说明

补记日志页面，展示本周不完整日志的日期列表，并提供针对性补记入口（体重、餐食、训练）。帮助用户快速识别遗漏的数据并高效完成补记。

## 关键组件

- 头部"补记日志"+返回箭头
- 英雄区: "本周有2天日志不完整"
- 缺失日志卡: 3月21日(缺训练), 3月20日(缺体重+餐食)
- 统计: 周完成度5/7天进度条, 连续记录12天徽章

## 入口

- history_list"补记日志"按钮

## 出口

- "补记训练" → start_workout_sheet
- "补记体重" → record_body_stats
- "补记餐食" → record_meal_sheet
- 返回 → history_list
