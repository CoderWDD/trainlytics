# 训练模板编辑器

所属模块: 模板 (Templates)
页面类型: 子页面

## 页面说明
训练模板编辑器，用于创建/编辑训练模板，包含名称、备注和动作列表（含组数/次数/重量配置）。

## 关键组件
- 头部+"返回模板列表"链接
- 模板名称输入("推日训练")
- 备注/目标文本区
- 动作列表(3个): 杠铃卧推(胸部, 复合)含2组(8次, 85%重量), 杠铃推举(肩部, 复合)含1组
- 各动作"添加组数"虚线按钮+删除按钮
- "添加动作"CTA卡
- 删除+"保存模板"按钮

## 入口
- template_gallery训练模板卡
- template_gallery"新建模板"
- start_workout_sheet"创建新模板"

## 出口
- "添加动作" → exercise_picker
- "保存模板" → template_gallery
- 删除 → template_gallery
