# 导出数据

所属模块: 设置 (Settings)
页面类型: 子页面

## 页面说明
数据导出配置页，用户选择导出格式、时间范围和内容类别，并在导出前预览摘要。

## 关键组件
- 头部"导出数据"+返回箭头
- 左侧: 导出格式切换(CSV/JSON)
- 时间范围按钮(7天/30天/全部/自定义)
- 内容复选框: 每日日志, 身体数据, 餐食, 训练
- 右侧: 导出摘要卡(1,284数据点, 2.4 MB, 本地加密存储)
- 隐私卡(AES-256加密)
- 同步卡(4台本地设备)
- "开始导出"渐变按钮

## 入口
- settings"导出数据"

## 出口
- "开始导出" → 触发下载
- 返回 → settings
