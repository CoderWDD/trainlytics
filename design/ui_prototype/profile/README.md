# 个人资料页

所属模块: 设置 (Settings)
页面类型: 子页面

## 页面说明

用户个人信息管理中心，展示和编辑头像、基础身体数据（身高/性别/年龄）、当前健康目标概览和使用统计（记录天数/连续天数/训练次数等）。

## 关键组件

- TopAppBar（返回 + "个人资料"标题 + 编辑按钮）
- 头像区（圆形头像 + 用户名 + 当前阶段标签）
- 基础信息卡（身高/性别/年龄/体型，可编辑）
- 当前目标概览卡（目标体重/当前体重/已达成进度条）
- 使用统计 Bento 卡片组：
  - 连续记录天数（大字体，徽章）
  - 总记录天数
  - 总训练次数
  - 累计训练量（kg）
- 身体数据历史区（最新体重/体脂/BMI，及上次变化）

## 入口

- settings "个人资料" 菜单项
- today_dashboard 头像点击

## 出口

- 返回 → settings 或上一页
- "编辑" → 进入编辑模式（内联编辑）
