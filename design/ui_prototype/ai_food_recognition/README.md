# AI食物识别

所属模块: 营养 (Nutrition) → AI识别
页面类型: 子页面

## 页面说明
AI食物识别结果页，展示从照片中识别出的食物及其营养成分，提供确认或微调选项。

## 关键组件
- 头部"AI识别结果"+返回箭头
- AI视觉画布: 三文鱼照片+扫描框叠加(角标+识别标签)
- 结果面板: "AI已识别"标签, "香煎三文鱼"标题, "微调"按钮
- 宏量Bento网格: 480kcal, 42g蛋白质, 250g重量, 28g脂肪
- 表现洞察卡: "84%每日蛋白质目标", Omega-3益处提示
- "确认并添加"渐变按钮

## 入口
- record_meal_sheet"分析餐食"
- food_search_sheet"AI识别"

## 出口
- "微调" → 编辑数值
- "确认并添加" → 保存到餐食记录
- 返回箭头
