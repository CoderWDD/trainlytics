# 餐食记录

所属模块: 今日 (Today) → 记录
页面类型: 子页面 + 底部弹窗

## 页面说明

餐食记录页面，展示每日摄入概要、水分和活动数据，底部弹窗提供AI扫描识别和手动宏量录入两种添加方式，满足快速记录与精确输入两种场景。

## 关键组件

- 每日摄入头部"1,840/2,400千卡"
- 水分卡(2.4L/3.5L, 68%), 活动消耗卡(482 kcal)
- 近期记录列表
- 底部弹窗"添加餐食": AI扫描文本区+"分析餐食"按钮, 宏量输入网格(卡路里/蛋白质/碳水/脂肪), "应用模板"下拉, "保存餐食记录"按钮

## 入口

- quick_add_sheet"添加餐食"
- today_dashboard营养卡

## 出口

- "分析餐食" → ai_food_recognition
- 手动录入 → manual_food_entry
- "保存餐食记录" → 保存并返回
