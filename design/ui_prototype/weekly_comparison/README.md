# 每周数据对比页

所属模块: 洞察 (Insights)
页面类型: 子页面

## 页面说明

多周横向对比视图，支持本周 vs 上周 vs 上周期（或自定义周）的体重、营养和训练数据对比，帮助用户评估阶段进展。

## 关键组件

- TopAppBar（返回 + "周对比"标题）
- 周选择器（最近4周的横向标签选择，可多选2-3周）
- 体重对比区块（多周体重折线图叠加，图例标注）
- 营养对比 Bento 卡组：
  - 平均热量对比（柱状对比）
  - 蛋白质达成率对比（环形图并列）
  - 营养目标达成天数对比
- 训练对比区块：
  - 训练频率对比（每周次数）
  - 总训练量对比（柱状）
  - 主要动作1RM对比（折线）
- 综合评分对比（表现分雷达图或柱状对比）
- 结论摘要区（AI生成的对比亮点文字，如"本周蛋白质达成率提升15%"）

## 入口

- insights_dashboard "周对比" 入口按钮
- weekly_review 页面跳转

## 出口

- 返回 → insights_dashboard
