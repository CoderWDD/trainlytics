# 选择动作

所属模块: 训练 (Workout)
页面类型: 底部弹窗 (Bottom Sheet)

## 页面说明

选择动作弹窗，从动作库中搜索、分类筛选并多选添加训练动作到当前训练或模板，支持快速搜索和按肌群分类浏览。

## 关键组件

- 深色背景
- 底部弹窗: "选择动作"标题, 关闭按钮
- 搜索栏
- 分类标签: 全部/胸部/背部/腿部/肩部/手臂/核心
- 动作卡片(含图片+复选框): 杠铃卧推, 哑铃飞鸟, 引体向上, 深蹲
- 底部固定栏: "添加所选(N)"渐变按钮

## 入口

- active_workout"添加动作"
- workout_template_editor"添加动作"

## 出口

- "添加所选" → 返回并添加动作
- 关闭 → 返回
