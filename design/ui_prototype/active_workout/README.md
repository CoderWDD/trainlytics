# 训练进行中

所属模块: 今日 (Today) → 训练中
页面类型: 子页面

## 页面说明

正在训练会话页面，用户在此实时记录每个动作的组数、次数、重量和RPE值。包含训练计时器和训练量洞察。

## 关键组件

- 训练计时器（00:45:23）
- 动作卡片含组数表格: 杠铃卧推（3组，重量/次数/RPE输入），杠铃推举（1组未完成）
- "添加组"按钮
- 训练洞察Bento: 估算训练量4,520 lbs，平均RPE 8.2
- 底部操作栏: "添加动作"，"完成训练"

## 入口

- `today_dashboard` "继续训练"
- `start_workout_sheet` 选择模板/空白训练

## 出口

- "添加动作" → `exercise_picker`
- "完成训练" → `today_dashboard`
