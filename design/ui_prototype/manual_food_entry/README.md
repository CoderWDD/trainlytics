# 手动食物录入

所属模块: 今日 (Today) → 记录
页面类型: 子页面

## 页面说明

手动食物录入表单，用于精确宏量营养素记录，用户输入食物名称、重量和各营养素数值，适合需要精确控制饮食数据的场景。

## 关键组件

- 头部"记录营养"/"LOG NUTRITION"
- "The Kinetic Lab"标签
- 表单: 食物名称输入, 重量(克)输入
- 宏量Bento网格: 卡路里(火焰图标), 蛋白质(健身图标), 碳水(谷物图标), 脂肪(水滴图标)
- "精准校对"信息框
- "保存到餐食"渐变按钮

## 入口

- record_meal_sheet手动输入
- food_search_sheet"手动输入"

## 出口

- "保存到餐食" → 返回record_meal_sheet
- 返回箭头
