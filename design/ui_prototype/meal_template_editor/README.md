# 餐食模板编辑

所属模块: 模板 (Templates)
页面类型: 子页面

## 页面说明

餐食模板编辑器，用于创建/编辑餐食模板，包含名称、总热量、宏量分解和食物条目列表，方便用户快速复用常见餐食组合。

## 关键组件

- 头部"编辑模板"+删除/保存按钮
- 模板名称输入("高蛋白早餐")
- 总能量输入(640 kcal)
- 宏量Bento卡: 蛋白质45g, 碳水52g, 脂肪18g
- 食物列表: 有机全蛋(3个, 210kcal), 希腊酸奶(200g, 120kcal), 即食燕麦(80g, 310kcal), 各有删除按钮
- "添加食物"按钮
- 备注区

## 入口

- template_gallery餐食模板卡

## 出口

- "添加食物" → food_search_sheet
- 保存 → template_gallery
- 删除 → template_gallery
