# 记录身体数据

所属模块: 今日 (Today) → 记录
页面类型: 子页面

## 页面说明

身体数据记录表单页，使用横向刻度尺选择器输入体重、体脂率和腰围数据。

## 关键组件

- 关闭按钮 + 时间戳头部
- 三个刻度尺选择器卡片: 体重（82.4 kg），体脂率（14.2%），腰围（84.5 cm）
- "保存记录"渐变主色按钮

## 入口

- `today_dashboard` "[记录]"
- `quick_add_sheet` "记录体重"
- `backfill_journal` "补记体重"

## 出口

- 关闭（X）→ 上一页
- "保存记录" → 保存并返回
