# 数据导入页

所属模块: 数据管理 (Data)
页面类型: 子页面

## 页面说明

从备份文件或第三方 CSV 恢复/迁移数据，包含文件选择、格式校验、数据预览和冲突处理流程。

## 关键组件

- TopAppBar（返回 + "数据导入"标题）
- 导入方式选择卡组（JSON备份恢复 / CSV格式迁移，2选1单选卡）
- 文件选择区（虚线边框拖拽区 + "选择文件"按钮）
- 文件预览卡（文件名 + 大小 + 包含数据类型标签）
- 数据校验进度条（校验中/校验成功/校验失败三态）
- 校验结果摘要（可导入条目数 + 冲突条目数）
- 冲突处理选项（保留本地 / 使用导入 / 逐条确认，单选）
- 导入进度弹窗（百分比 + 当前处理项）
- PrivacyCard（数据安全说明）
- 主色渐变"开始导入"按钮

## 入口

- settings "数据导入" 菜单项
- export_data 页面底部链接

## 出口

- 导入完成 → settings（含成功 Snackbar）
- 返回 → settings
