# 单日详情

所属模块: 历史 (History)
页面类型: 子页面

## 页面说明

单日详细表现回顾页，展示该日的身体数据、训练动作详情（含训练量和PR标记）、餐食明细和宏量营养素目标进度。

## 关键组件

- 警告横幅"您正在查看过去的记录"
- Bento网格:
  - 身体成分卡（82.4kg）
  - 训练卡"大重量下肢-肥大训练"（深蹲5组140kg PR，罗马尼亚硬拉3组100kg，腿举4组55kg）
- 营养区: 餐食列表（练后餐620kcal，蛋白粉310kcal，三文鱼晚餐740kcal）
- 宏量进度条: 蛋白质195g/200g（97%），碳水240g/300g（80%），脂肪65g/70g（92%）

## 入口

- `history_list` 点击某天卡片

## 出口

- 返回 → `history_list`
- 底部导航切换
