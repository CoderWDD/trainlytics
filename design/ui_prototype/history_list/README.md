# 历史记录

所属模块: 历史 (History)
页面类型: 主页面 (Tab页)

## 页面说明

历史记录主页面，展示周日历条和每日活动卡片时间线，包含完成状态标记。

## 关键组件

- 头部"历史记录" + "2026年3月 表现日志"
- 水平日历条（3月16-22日），颜色编码圆点（绿色=已记录，边框=部分，红色=未完成）
- 时间线卡片: 各日的体重、餐食、训练摘要
- "补记日志"CTA

## 入口

- 底部导航"历史"Tab

## 出口

- 点击日期卡 → `history_day_detail`
- "补记日志" → `backfill_journal`
- 底部导航切换
