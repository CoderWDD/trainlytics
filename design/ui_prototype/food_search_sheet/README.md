# 食物搜索

所属模块: 模板 (Templates)
页面类型: 底部弹窗 (Bottom Sheet)

## 页面说明
食物搜索弹窗，从食物数据库中搜索和选择食物添加到餐食模板，支持手动输入和AI识别两种方式。

## 关键组件
- 背景: 模板视图
- 底部弹窗"添加食物": 搜索栏
- 分类标签: 常用/肉类/蛋奶/五谷/果蔬/零食
- 食物卡片(含图片+加号): 全麦面包(100g, 247kcal), 鸡胸肉(100g, 165kcal), 水煮蛋(1个, 78kcal), 脱脂牛奶(250ml, 88kcal)
- 底部: "手动输入"/"AI识别"分列按钮

## 入口
- meal_template_editor"添加食物"

## 出口
- "+"添加食物
- "手动输入" → manual_food_entry
- "AI识别" → ai_food_recognition
- 关闭弹窗
