# Trainlytics

本地优先的Android减脂追踪应用

## 项目简介

Trainlytics 是一款专为减脂期设计的本地优先健身追踪应用，帮助用户记录体重、餐食、训练和恢复数据，并提供轻量级的趋势分析和周回顾功能。

## 核心功能

- **今日记录**: 集中管理每日身体数据、餐食、训练和笔记
- **AI餐食识别**: 拍照即可自动识别食物和营养成分
- **训练追踪**: 记录训练动作、组数、重量和力量进展
- **数据分析**: 体重趋势、营养摄入、训练量等可视化分析
- **周回顾**: 每周进展总结和对比
- **模板系统**: 表单模板和内容模板提高录入效率
- **数据导出**: 支持CSV和JSON格式导出

## 技术特性

- 本地优先存储，无需账号和云同步
- 离线完全可用
- Material Design 3 设计语言
- Android原生开发
